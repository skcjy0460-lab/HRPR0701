# 🏥 병원 인사노무 실무 가이드 시스템

> 병원 개원·경영 컨설턴트 및 인사노무 실무 담당자를 위한 종합 가이드 시스템

---

## 📦 파일 구성

| 파일명 | 설명 |
|---|---|
| `hospital_hr_guide.py` | 메인 앱 (사이드바, 라우팅, CSS) |
| `job_description.py` | 직무기술서(JD) 작성 모듈 |
| `onboarding_guide.py` | 직원 온보딩 가이드 모듈 |
| `ai_consulting.py` | AI 인사노무 상담 모듈 (OpenAI API 연동) |
| `hr_calculator.py` | 인사노무 스마트 계산기 모듈 |
| `requirements.txt` | 필요 패키지 목록 |

---

## 🚀 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. OpenAI API 키 설정 (AI 상담 기능 사용 시)
```bash
export OPENAI_API_KEY="your-api-key-here"
# 또는 .env 파일에 저장
```

### 3. 앱 실행
```bash
streamlit run hospital_hr_guide.py
```

## 🎯 주요 기능

### 1. 📝 직무기술서(JD) 작성
- **병원 직종별 템플릿 제공**: 간호사(병동/외래/ICU), 의사, 물리치료사, 방사선사, 임상병리사, 원무팀, 원장 등 15개 직종
- **커스터마이징 가능**: 병원명, 부서, 직급, 담당업무, 자격요건 등 직접 입력
- **HTML 다운로드**: 완성된 직무기술서를 HTML 파일로 다운로드 (인쇄 최적화)
- **미리보기 기능**: 저장 전 최종 확인

### 2. 🚀 직원 온보딩 가이드
- **입사 전/1일차/1주차/1개월차/3개월차** 단계별 체크리스트
- **직종별 맞춤 온보딩**: 간호사, 의사, 행정직 등 직종에 따른 차별화된 가이드
- **병원 특화 항목**: 감염관리 교육, 의료기기 사용법, EMR 시스템 교육 등
- **체크리스트 인쇄**: 담당자용 체크리스트 출력 기능

### 3. 🤖 AI 인사노무 상담
- **병원 특화 AI**: 근로기준법, 4대보험, 의료법, 법정의무교육 전문 답변
- **자주 묻는 질문 빠른 선택**: 10가지 실무 질문 원클릭 입력
- **2026년 최신 기준 적용**: 최저임금, 4대보험 요율 등 최신 법령 반영
- **대화 히스토리**: 연속 질문 및 맥락 유지 기능

### 4. 🧮 인사노무 스마트 계산기

#### 💰 퇴직금 계산
- 평균임금 자동 계산 (3개월 임금 기준)
- 연차수당 포함 여부 선택
- DC형/DB형 퇴직연금 안내
- 중간정산 가능 여부 체크

#### 🌴 연차 계산
- 1년 미만 월별 연차 자동 계산
- 1년 이상 근속별 연차 산정 (최대 25일)
- 연차수당 계산 (미사용 연차 × 통상일급)
- 연차 촉진 제도 안내

#### 🏖️ 주휴수당 계산
- 주 소정근로시간 입력 → 주휴수당 자동 계산
- 단시간 근로자 주휴수당 산정
- 주휴수당 포함 최저임금 검증

#### 🛡️ 4대보험 계산
- 2026년 최신 요율 적용 (국민연금 9.5%, 건강보험 7.19%, 장기요양 13.14%, 고용보험 1.8%)
- 근로자/사업주 부담분 분리 표시
- 사업장 규모별 고용보험 차등 적용
- 월 실수령액 추정

#### ⏱️ 가산수당 계산
- **연장근로수당**: 법정근로시간 초과 × 통상시급 × 1.5배
- **야간근로수당**: 22:00~06:00 × 통상시급 × 0.5배 (기본급 별도)
- **휴일근로수당**: 8시간 이내 1.5배, 8시간 초과 2.0배
- **중복 적용**: 야간+연장, 야간+휴일 중복 시 자동 합산
